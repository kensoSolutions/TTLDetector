# TTLDetector
![Plugin image](https://github.com/kensoSolutions/TTLDetector/blob/readme-edit/plugin.jpg "Plugin image")  
TTL detector plugin for the open-ephys [plugin-GUI](https://github.com/open-ephys/plugin-GUI/ "pluguin-GUI"). 
## Installation
Copy the BinaryReader folder to the plugin folder of your GUI. Then build the plugin as described in the [wiki](https://open-ephys.atlassian.net/wiki/spaces/OEW/pages/491544/Installation "wiki").
